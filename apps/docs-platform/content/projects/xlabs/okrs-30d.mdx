## O1 — Factory / Operating System

Set up the infrastructure, processes, and tooling to operate XLabs from day 1.

| Key Result | Target | Frequency |
|-----------|--------|-----------|
| Standardized tech stack deployed | 1 stack operational | Weekly |
| 3 agents operational (Orchestrator, PRD, Deck) | 3 agents | Weekly |
| Playbook documented (rituals, checklists, templates) | 1 playbook | Monthly |
| Structured project folder template | 1 template | Weekly |

Recruit and onboard the first cohort of interns.

| Key Result | Target | Frequency |
|-----------|--------|-----------|
| 20 candidates sourced | 20 | Weekly |

| 3 interns onboarded (7-day program) | 3 | Monthly |
| 100% have assigned project with PRD | 100% | Weekly |
