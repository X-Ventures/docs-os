# Catalyst — Product Vision

> The deal flow OS for Web3 venture capital.

---

## Core Product: PitchScan AI

Upload any pitch deck (PDF) and get a comprehensive analysis from **10 simulated VC personas** in under 10 seconds.

| Analysis Layer | What It Does |
|---------------|-------------|
| **Key Info Extraction** | OCR + AI extracts company name, raise, valuation, vertical, team, socials |
| **VC Committee** | 10 personas evaluate independently (seed specialist, growth VC, technical VC, etc.) |
| **Consensus Score** | 0-10 weighted score with majority vote: INVEST / PASS / CONSIDER |
| **Deep Reasoning** | 6-dimension analysis: Problem/Solution, Team, Market Size, Market Interest, Competition, GTM |
| **Highlights & Risks** | Key strengths and red flags surfaced automatically |
| **Meta Analysis** | Committee synthesis with actionable recommendation |

**Parameters**: model gpt-4.1-mini, reasoning_effort medium, include_due_diligence true, max_personas 10.

## Platform Architecture

**4 Input Sources** feed into a unified PitchScan pipeline:

| Source | Entry Point | Flow |
|--------|------------|------|
| Startup Dashboard | `/dashboard/startup/pitch-decks` | Upload PDF, async analysis, publish to deal flow |
| Investor Analytics | `/dashboard/investor/analytics` | Upload PDF, instant 10-persona analysis |
| Telegram Bot | Group message with DocSend URL | Auto-detect, download via docsend2pdf, analyze, post results |
| Admin Dashboard | Internal | Manual trigger, full access |

All sources create: DealQueue entry (tracking) + Startup draft (pipeline) + Analysis record.
